# Welcome to the IQSS Archive

We're excited to introduce this comprehensive archive of the Harvard Institute for Quantitative Social Science website. This Hugo-powered site contains a complete snapshot of IQSS content, including:

## What's Included

- **1,075+ People Profiles** - Complete directory of IQSS faculty, staff, affiliates, and students
- **Research Programs** - Information about ongoing initiatives and projects  
- **Data Science Services** - Resources for statistical consulting and support
- **Technology Services** - Computing facilities and technical resources
- **News & Events** - Latest updates and announcements
- **Historical Content** - Archived materials for research and reference

## Interactive Features

This site includes several modern features not available in the original:

### Giscus Comments
Each blog post supports interactive discussions powered by GitHub Discussions. You can:
- Leave comments and feedback
- Start conversations about research topics
- Engage with the IQSS community

### Enhanced Search
The Hugo-powered search makes it easy to find specific content across all sections.

### Mobile-Responsive Design
The site works seamlessly across all devices and screen sizes.

## Technical Details

Built with:
- **Hugo Static Site Generator** - Fast, secure, and SEO-friendly
- **Ananke Theme** - Clean, academic-focused design
- **GitHub Pages** - Reliable hosting with automatic deployments
- **Giscus** - GitHub Discussions-powered comments

## Educational Purpose

This archive serves as an educational resource and research tool. All content is sourced from the official IQSS website (iq.harvard.edu) and is preserved for academic and research purposes.
