KVM Access to Servers

Remotely via a web browser -- This is only working with older version of the Java JRE (<=1. 6. 0. 10 on Windows and <=1. 6. 0. 7 on Linux) as of 2009-05-14. Open a web browser and navigate to https://dsview. hmdc. harvard. edu/. Log in using your LDAP username and password.
