<div class="hwp-text-block field field--name-field-hwp-body field--type-text-long field--label-hidden"><p>	Confidential information must be encrypted whenever it is sent over any network. Never send confidential information unencrypted in e-mail, even if the connection to the e-mail server is encrypted.  High-risk confidential information may be sent only directly from one approved system to another.<br/><br/>If you are transferring files containing confidential information or if your files are simply too large to send over email, please use the Kiteworks (Accelion) Secure File Transfer System.  The Kiteworks server provides robust data security and accommodates the transfer of files and folders up to 45 GB's in size. More information can be found here: <a href="https://harvard.service-now.com/ithelp?id=sc_cat_item&amp;sys_id=06841947dbcf130043f21bfa4b9619eb">https://harvard.service-now.com/ithelp?id=sc_cat_item&amp;sys_id=06841947dbc...</a></p><p>	For help with secure file transfer, please <a href="mailto:help@iq.harvard.edu">contact us</a>.</p></div>
