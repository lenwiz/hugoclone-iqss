<div class="hwp-text-block field field--name-field-hwp-body field--type-text-long field--label-hidden"><p>To access your network space while off campus:<br/>Install and configure Okta app on your smart device if you haven't done so already. Details can be found at <a href="https://key.harvard.edu/">https://key.harvard.edu</a><br/>Install Harvard's VPN Client:<br/> You can obtain this software from <a href="http://vpn.harvard.edu/">http://vpn.harvard.edu</a><br/>Log in with your Harvard username and Key and additional code from the Okta app. Download and install the Cisco VPN application. When opening the application for the first time, the address to connect to is vpn.harvard.edu/gencom</p><p><br/>click <strong>connect</strong> and enter your key information and additional code from Okta.</p><p><br/>click <strong>OK</strong> and then you will be asked to acknowledge that the system is for Harvard associates only.</p><p> </p><p>If you accept, the connection process will continue. Once connected, you will be able to connect to departmental shares and other Harvard resources. </p></div>
