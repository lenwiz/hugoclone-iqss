Zoom for Teaching, Workshops and Conferences

There are a substantial number of guides that will teach you the mechanics of using Zoom for video conferencing, web conferencing, and webinars. What is missing, though, is a guide that answers the question, “now that I know how to use Zoom, how can I use it to teach, workshop, or hosting a large conference?” That is where this guide comes in.
