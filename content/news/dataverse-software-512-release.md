The Dataverse Project is pleased to announce the release of version 5.12, delivering a comprehensive set of new features, enhancements, and bug fixes that improve the research data repository experience for both users and administrators. This release continues the project's commitment to regular, high-quality updates that respond to the needs of the global Dataverse community.

Version 5.12 introduces several highly requested features that enhance data discovery, access management, and metadata handling. Improvements to the user interface make common workflows more intuitive, while new administrative tools give repository managers better control over their collections and improved insight into repository usage. Enhanced API capabilities expand integration possibilities with other research tools and institutional systems, supporting more seamless data management workflows across the research lifecycle.

The release also addresses numerous bugs identified by community members, improving system stability and resolving issues that affected specific use cases. Performance optimizations ensure that Dataverse repositories continue to operate efficiently as data collections grow. Repository administrators are encouraged to review the detailed release notes to understand the full scope of improvements and plan their upgrade accordingly. The collaborative, open-source development model continues to make Dataverse the platform of choice for hundreds of institutions worldwide committed to open science and reproducible research.
