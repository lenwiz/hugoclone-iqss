The Dataverse Project is excited to announce the arrival of version 6.0, a major milestone release that brings comprehensive platform upgrades to ensure the software runs on modern, well-supported infrastructure components. This significant update includes upgrades to Payara application server, Solr search engine, and Java runtime environment, positioning Dataverse repositories on a solid foundation for continued growth and development.

The infrastructure upgrades in version 6.0 provide critical improvements to security, performance, and long-term maintainability. Moving to current versions of Payara, Solr, and Java ensures that Dataverse installations benefit from the latest bug fixes, security patches, and performance optimizations available in these core technologies. These platform updates also enable future feature development by providing access to modern capabilities and APIs that weren't available in older versions. While platform upgrades require careful planning and testing, they are essential for maintaining a healthy, sustainable software ecosystem.

Repository administrators should note that this major version release may require more extensive upgrade procedures than typical point releases. The Dataverse team has prepared comprehensive upgrade documentation and migration guides to support smooth transitions. Testing in staging environments is strongly recommended before upgrading production systems. The 6.0 release represents a significant investment in the long-term health and sustainability of the Dataverse platform, ensuring it remains the trusted choice for research data repositories worldwide for years to come.
