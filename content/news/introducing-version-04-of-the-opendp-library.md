The OpenDP Project is pleased to announce the release of version 0.4 of the OpenDP Library, introducing important new measurements and significant floating point improvements that enhance both the functionality and reliability of the differential privacy toolkit. This release represents substantial progress toward making differential privacy practical and trustworthy for protecting sensitive data in real-world applications.

Version 0.4 expands the library's capabilities with additional measurements that enable new types of differentially private analyses. These new statistical tools provide data practitioners with more options for extracting insights from sensitive datasets while maintaining rigorous privacy guarantees. The floating point improvements address critical technical challenges in implementing differential privacy algorithms, ensuring that computations maintain accuracy and reliability even when dealing with numerical precision issues that can arise in complex calculations.

The floating point enhancements are particularly important for building trustworthy differential privacy systems, as even small numerical errors can potentially compromise privacy guarantees. By carefully addressing these technical details, OpenDP aims to provide implementations that practitioners can confidently deploy in production systems handling sensitive data. The release continues OpenDP's trajectory toward becoming the standard open-source toolkit for differential privacy, combining mathematical rigor with practical usability to enable responsible data analysis and sharing across sectors.
