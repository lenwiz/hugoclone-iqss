The Dataverse Project announces the release of version 5.14, bringing an extensive collection of new features, enhancements, and bug fixes that strengthen the platform's capabilities as a comprehensive research data repository solution. This release reflects ongoing collaboration among developers, repository administrators, and researchers worldwide who contribute to making Dataverse more powerful and user-friendly.

Version 5.14 introduces meaningful improvements across multiple areas of the platform. Enhanced metadata management features provide researchers with more flexible ways to describe their datasets, improving discoverability and promoting best practices in data documentation. New visualization and preview capabilities make it easier for users to understand dataset contents before downloading, while improved access controls give data owners more granular options for managing who can view and use their research outputs. API enhancements expand programmatic access to repository functions, supporting automation and integration with institutional research infrastructure.

Bug fixes in this release address issues reported by the active Dataverse community, improving reliability and resolving edge cases that affected specific workflows. Performance optimizations ensure the platform continues to scale effectively as repository collections expand. The release demonstrates the vitality of the Dataverse open-source ecosystem, where institutions worldwide pool resources and expertise to build infrastructure that advances open science, supports reproducible research, and ensures long-term preservation of valuable scholarly data.
