The Dataverse Project is excited to announce that version 6.2 is now available, bringing a valuable collection of updates and new features that enhance the platform's capabilities for managing and sharing research data. This release continues the steady evolution of the 6.x series, delivering improvements that respond to community feedback and emerging needs in the research data management landscape.

Version 6.2 introduces features that improve both the researcher experience and administrative functionality. Enhanced data management tools make it easier to organize collections and apply appropriate access controls, while improvements to metadata handling support more comprehensive documentation of research outputs. New capabilities for data visualization and preview help users understand dataset contents more effectively, promoting data reuse and discovery. Administrative enhancements provide repository managers with better tools for monitoring system health, managing user permissions, and configuring repository behavior to meet institutional requirements.

The release also addresses bugs and performance issues identified by the community, improving stability and efficiency across various workflows. These improvements ensure that Dataverse repositories continue to operate reliably as they grow in size and usage. Repository administrators are encouraged to review the release notes for detailed information about changes and upgrade procedures. The 6.2 release reflects the ongoing commitment of the Dataverse community to building robust, user-friendly infrastructure that advances open science and supports the long-term preservation of scholarly data.
