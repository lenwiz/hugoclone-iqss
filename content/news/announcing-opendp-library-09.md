The OpenDP Project is proud to announce the release of OpenDP Library 0.9, a major milestone that expands the modular collection of differentially private algorithms available for building robust privacy-preserving applications. This release brings powerful new capabilities to developers and data scientists who need to extract insights from sensitive data while providing mathematically rigorous privacy guarantees.

Version 0.9 significantly expands the library's algorithmic toolkit, adding new measurements, transformations, and composition mechanisms that enable more sophisticated privacy-preserving analyses. The modular architecture allows users to construct custom differentially private pipelines by combining building blocks in flexible ways, each with formal privacy proofs that ensure the overall system maintains its privacy guarantees. Enhanced support for common data analysis tasks makes it easier to adopt differential privacy without requiring deep expertise in the underlying mathematics.

This release also includes substantial improvements to usability, performance, and stability based on extensive feedback from the growing OpenDP community. Documentation enhancements, additional code examples, and better error messages help new users get started quickly while supporting advanced use cases for experienced practitioners. The OpenDP team remains committed to building trustworthy open-source infrastructure that enables organizations across sectors to responsibly analyze and share insights from sensitive data while protecting individual privacy.
