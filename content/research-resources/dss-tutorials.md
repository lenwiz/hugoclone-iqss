## LaTeX and Writing Tools

- [LaTeX Applications: A Beginner's Guide](https://iqss.github.io/latex_guide/) — Covers recommended LaTeX editing options including Overleaf, VSCode, and Emacs, plus tools like GitHub and Copilot.
- [Overleaf: A Beginner's Guide](https://iqss.github.io/overleaf_guide/) — Steps for signing up, editing documents, and collaboration in Overleaf.
- [Visual Studio Code: A Beginner's Guide](https://iqss.github.io/vscode-settings/) — Steps for installation and importing recommended settings, keybindings, and shortcuts.
- [Emacs: A Beginner's Guide](https://iqss.github.io/emacs_guide/) — Steps for installation, importing configuration, and common functions for LaTeX editing.

## Version Control

- [Git and GitHub: A Beginner's Guide](https://iqss.github.io/git_guide/) — Steps for creating and cloning repositories, and using the command line.
- [GitHub Pages: A Beginner's Guide](https://iqss.github.io/github_pages_guide/) — Steps for creating a personal website, downloading and customizing templates.

## Data Science Methods

- [Rbuild: Create Your Own Packages in R](https://iqss.github.io/dss-rbuild) — Best practices for R package development and version control using RStudio.
- [Matching and Weighting for Causal Inference](https://iqss.github.io/dss-ps) — Covers propensity score analysis methods for adjusting confounding in observational studies. Includes R examples.
- [Web-scraping with Selenium Python](https://iqss.github.io/dss-webscrape/) — Guide to scraping both static and dynamic websites with Selenium Python.
