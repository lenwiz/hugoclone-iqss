We offer a series of data analysis and programming workshop materials to members of the Harvard and MIT communities, as well as to the public. Each workshop is self-contained and can be studied at one's own pace.

## Python Workshop Materials

### Python Introduction

This hands-on workshop introduces the basic elements of Python by working through an example of how to analyze text data. Appropriate for those with little or no prior experience.

- [Setup instructions](https://iqss.github.io/dss-workshops/PythonInstall.html)
- [All workshop materials (zip)](https://github.com/IQSS/dss-workshops/raw/master/Python/PythonIntro.zip)
- [Static workshop notes](https://iqss.github.io/dss-workshops/PythonIntro.html)

### Python Web Scraping

This hands-on workshop introduces basic techniques for web-scraping using popular Python libraries. Intermediate-level for those who have used Python for at least a few months.

- [Setup instructions](https://iqss.github.io/dss-workshops/PythonInstall.html)
- [All workshop materials (zip)](https://github.com/IQSS/dss-workshops/raw/master/Python/PythonWebScrape.zip)
- [Static workshop notes](https://iqss.github.io/dss-workshops/PythonWebScrape.html)

## R Workshop Materials

### R Introduction

This hands-on workshop teaches how to use R to import and manipulate data, install packages, conduct basic analyses, and create graphical displays. Appropriate for those with little or no prior experience.

- [Setup instructions](https://iqss.github.io/dss-workshops/Rinstall.html)
- [All workshop materials (zip)](https://github.com/IQSS/dss-workshops/raw/master/R/Rintro.zip)
- [Static workshop notes](https://iqss.github.io/dss-workshops/Rintro.html)

### R Regression Models

Demonstrates how to deploy statistical procedures including multiple regression, modeling with categorical variables, and model diagnostics. Intermediate-level.

- [Setup instructions](https://iqss.github.io/dss-workshops/Rinstall.html)
- [All workshop materials (zip)](https://github.com/IQSS/dss-workshops/raw/master/R/Rmodels.zip)
- [Static workshop notes](https://iqss.github.io/dss-workshops/Rmodels.html)

### R Graphics

Introduction to the popular ggplot2 R graphics package, covering layering, mapping variables to aesthetics, scales, faceting, and themes. Intermediate-level.

- [Setup instructions](https://iqss.github.io/dss-workshops/Rinstall.html)
- [All workshop materials (zip)](https://github.com/IQSS/dss-workshops/raw/master/R/Rgraphics.zip)
- [Static workshop notes](https://iqss.github.io/dss-workshops/Rgraphics.html)

### R Data Wrangling

Prepares you for dealing with messy data by walking through a real-life example. Intermediate-level.

- [Setup instructions](https://iqss.github.io/dss-workshops/Rinstall.html)
- [All workshop materials (zip)](https://github.com/IQSS/dss-workshops/raw/master/R/RDataWrangling.zip)
- [Static workshop notes](https://iqss.github.io/dss-workshops/RDataWrangling.html)
