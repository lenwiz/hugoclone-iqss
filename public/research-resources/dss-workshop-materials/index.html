Data Science Services Workshop Materials

Home › Research Resources › Workshop Materials

We offer a series of data analysis and programming workshop materials to members of the Harvard and MIT communities, as well as to the public. Each workshop is self-contained and can be studied at one’s own pace.

Python Workshop Materials

Python Introduction

This hands-on workshop introduces the basic elements of Python by working through an example of how to analyze text data. Appropriate for those with little or no prior experience.

  • Setup instructions
  • All workshop materials (zip)
  • Static workshop notes

Python Web Scraping

This hands-on workshop introduces basic techniques for web-scraping using popular Python libraries. Intermediate-level for those who have used Python for at least a few months.

  • Setup instructions
  • All workshop materials (zip)
  • Static workshop notes

R Workshop Materials

R Introduction

This hands-on workshop teaches how to use R to import and manipulate data, install packages, conduct basic analyses, and create graphical displays. Appropriate for those with little or no prior experience.

  • Setup instructions
  • All workshop materials (zip)
  • Static workshop notes

R Regression Models

Demonstrates how to deploy statistical procedures including multiple regression, modeling with categorical variables, and model diagnostics. Intermediate-level.

  • All workshop materials (zip)
  • Static workshop notes

R Graphics

Introduction to the popular ggplot2 R graphics package, covering layering, mapping variables to aesthetics, scales, faceting, and themes. Intermediate-level.

R Data Wrangling

Prepares you for dealing with messy data by walking through a real-life example. Intermediate-level.
